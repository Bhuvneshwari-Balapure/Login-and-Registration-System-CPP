# Login and Registration System (C++)

A simple console-based user authentication system implemented in C++.  
This project allows users to register with a username and password, and later log in using the same credentials.  
All data is safely stored in text files using file handling.

## Features
- Register new users with unique usernames
- Secure login system with credential verification
- User data stored locally and persistently using file handling
- Validation checks for empty inputs and duplicate usernames
- Simple and interactive console-based UI

## Technologies Used
- C++ Programming Language
- File Handling (`fstream`)
- String Manipulation
- Standard Input/Output

## File Structure
- `main.cpp` — Source code for the login/registration system
- `users.txt` — Stores usernames and passwords
